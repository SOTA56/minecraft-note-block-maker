# 音符ブロックメーカー

Minecraftの音符ブロック演奏をスマートフォン上で設計するWebツール。時間軸が上から下へ進む縦ピアノロールを採用する。

## 開発

```sh
npm install
npm run dev
```

## 現在のMVP

- 25音・64ステップの縦ピアノロール
- 8トラック、1トラック1音色、和音入力
- ゴースト表示、音量、ミュート
- 最大同時発音数の表示
- 20 TPSを基準にしたWeb Audio試聴
- ブラウザ内自動保存と `.nbm` 書き出し・読み込み

`.nbm` の内容はバージョン付きJSON。回路生成は後続フェーズで実装する。
